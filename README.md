# [Curl.gl](https://darshankalita.github.io/Curl.gl/)
A tool to help visualize vector fields and their curl.

*Heavily inspired by [Andrei Kashcha's Fieldplay](https://github.com/anvaka/fieldplay).*

Try constraining this field to a plane! The simulation shows beautiful swirls. 

If you view a preset attractor, it might not have uniform particle generation in space since otherwise it becomes difficult to see it in full glory. Especially the Aizawa Attractor. The hover spawning feature is very useful here if you want to visualise custom strange attractors. Remember to turn it on from the control panel!

Another favourite of mine is Taylor-Greens vortex. You have to again use hover spawning but it simulates absolutely beautiful ice cube like visuals! 

Hopf Fibration is, in my opinion, the most difficult among these to get it to look right, but once it's done, it's worth it. 

Adjust velocity, lifetime, density, scale and even opacity according to the field for best viewing.
