# [Curl.gl](https://darshankalita.github.io/Curl.gl/)
A tool to help visualize vector fields and their curl.

Find it here: [Curl.gl](https://darshankalita.github.io/Curl.gl/)
*Heavily inspired by [Andrei Kashcha's Fieldplay](https://github.com/anvaka/fieldplay).*

If you view a preset attractor, it might not have uniform particle generation in space since otherwise it becomes difficult to see it in its glory. Especially the Aizawa Attractor. The hover spawning feature is very useful here if you want to visualise custom strange attractors.
 
Adjust velocity, lifetime and density according to the field for best viewing.
